# Chess

This is a GUI-based Chess game written primarily in C, designed as a reference for research or projects. The application offers a classic chess-playing experience with a graphical interface, making it suitable for learning, demonstration, or further development.

## Features

- **Complete Chess Game:** Play the full game of chess with all standard rules implemented.
- **Graphical User Interface:** Intuitive GUI for easy game play and interaction.
- **Research Friendly:** The codebase is structured for those who want to study, modify, or extend chess logic in C.
- **Educational Resource:** Suitable as a base for academic projects or AI enhancements.
